小红书开播检测

配置面板 · 填好后一键保存到 GitHub Secrets

⚠️ Token 只在你浏览器里用一次,不会上传到任何服务器。用完建议去 GitHub 撤销 token。

1填入 GitHub Token

需要 repo 权限(经典 token 勾选 repo 即可)。获取:github.com/settings/tokens

2填写监控配置

打开小红书网页版,进入要监控的用户主页,复制地址栏链接
Bark App 首页顶部显示的字符串,如 XXXXXXXXXX
用官方服务器就填 https://api.day.app,自建的填自建地址
不填可能检测不到直播状态。整行粘贴即可。

3立即测试一次

保存配置后,点下面按钮手动触发一次检测,看手机 Bark 是否收到推送(当前没开播则不会推送,可去 Actions 看日志)。

配置仅保存在 GitHub Secrets(加密),不会出现在公开代码里。
Actions 每 10 分钟自动检测一次 · 检测到开播即推送 Bark
